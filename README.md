贡献
----

如果你对该项目有兴趣，欢迎你为本项目贡献。主要有两方面的工作：
* 翻译
* 纠错

如果你选择翻译，请先在issue中提出你要翻译的文件，翻译完成后提出pull request请求即可。

如果你选择纠错，那么直接在issue中提出即可。

本地预览
-------
出于各种原因，你需要在本地就行预览。

请参考gitbook工具文档，安装并启动本地服务器。

```
npm install -g gitbook
gitbook serve libgdx-book
```
